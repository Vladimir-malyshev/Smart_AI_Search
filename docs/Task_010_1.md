Архитектурная концепция модуля:
Модуль принимает список текстовых запросов, параллельно выполняет поиск через выбранного провайдера, нормализует результаты в единый формат и отдает дедуплицированный список сниппетов. В модуле должен быть реализован паттерн "Роутер" (Feature Flag), чтобы мы могли переключать поисковые движки через переменные окружения без изменения кода.
