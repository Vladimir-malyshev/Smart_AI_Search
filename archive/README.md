# Архив модулей

Эта директория содержит устаревшие или временно отключенные модули и тесты, которые были удалены из активного пайплайна в ходе рефакторинга архитектуры. Код перемещен сюда для сохранения истории и заложенных идей.

## `search_router.py`
Устаревший модуль маршрутизации поисковых систем (Jina, DDG, SearXNG). 
Причина перемещения: полностью заменён новой реализацией в `execution_engine.py`. Содержал дублирующую логику и устаревший датакласс `SearchSnippet`.

## `query_expansion.py`
Модуль для генерации набора поисковых запросов из исходного пользовательского намерения.
Причина перемещения: данный этап был интегрирован в `ai_judge.py`, который теперь сам расширяет запросы, если считает собранную информацию недостаточной. Первичный запрос обрабатывается "как есть", чтобы предотвратить потерю смысла до первого сбора информации (согласно Task_011).

## Тесты
Файлы `test_search_router.py` и `test_query_expansion.py` перенесены сюда для сохранения тестовых сценариев к устаревшим модулям.
