    Пожалуйста, обнови файлы app/modules/jina_reader.py и app/modules/execution_engine.py:

    1. Тюнинг Ридера (app/modules/jina_reader.py)

        В запросах к s.jina.ai добавь заголовок "X-Engine": "google". Это обеспечит наиболее релевантную выдачу по сравнению с дефолтными движками.

        Внедрите фильтрацию по языку: если поисковый запрос содержит кириллицу, автоматически добавляй к строке запроса оператор  lang:ru (например, через f-строку или простую проверку символов). Это отсечет случайный иностранный мусор в выдаче.

        Убедись, что GLOBAL_TIMEOUT_SEC установлен на 200.0, чтобы сложные запросы не прерывались на середине второй итерации.

    Изменяй только указанные файлы.

Почему это важно:

    X-Engine: google: По умолчанию Jina может использовать более дешевые поисковые движки. Переключение на Google — это переход в «высшую лигу» по качеству ссылок.

    lang:ru: Это «серебряная пуля» против дорвеев. Вместо того чтобы Гемма мучилась, выбирая между плохой русской ссылкой и плохой индийской, мы просто не покажем ей вторую.

    X-Retain-Links: none: Это спасет нас от «мусора» в r.jina. В статьях бывает сотни ссылок на «Похожие материалы», которые для Судьи — просто белый шум.
